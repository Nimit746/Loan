## ğŸ“¦ Installation

### Prerequisites

Before you begin, ensure you have the following installed:

```bash
# Check Node.js version (v18 or higher required)
node --version

# Check Python version (3.9 or higher required)
python --version

# Check Git
git --version
```

**Required Software:**
- [Node.js](https://nodejs.org/) (v18+)
- [Python](https://www.python.org/downloads/) (3.9+)
- [Git](https://git-scm.com/downloads)

### Quick Start Guide

#### Step 1: Clone the Repository

```bash
git clone https://github.com/Nimit746/loanai.git
cd loanai
```

#### Step 2: Frontend Setup

```bash
# Navigate to client directory
cd client

# Install dependencies
npm install

# Start development server
npm run dev
```

âœ… Frontend will be running at `http://localhost:5173`

#### Step 3: Backend Setup

Open a new terminal window:

```bash
# Navigate to server directory
cd server

# Create virtual environment (recommended)
python -m venv venv

# Activate virtual environment
# On Windows:
venv\Scripts\activate
# On macOS/Linux:
source venv/bin/activate

# Install dependencies
pip install -r requirements.txt

# Start FastAPI server
python app.py
```

âœ… Backend API will be running at `http://localhost:8000`

#### Step 4: Verify Installation

Visit `http://localhost:5173` in your browser. You should see the LoanAI landing page.

### Alternative: Concurrent Execution

Run both frontend and backend simultaneously with a single command:

```bash
# From root directory
npm install
npm run dev
```

This requires the root `package.json` to have concurrently configured.

## ğŸ§ª Model Training

### Training New Models

#### Prepare Your Dataset

Ensure your CSV file has the following columns:
```
Income, Age, Experience, Married, House_Ownership, Car_Ownership,
Profession, CITY, STATE, CURRENT_JOB_YRS, CURRENT_HOUSE_YRS, Risk_Flag
```

#### Run Training Script

```bash
cd server/training

# Train all models
python train_models.py --dataset ../data/loan_data.csv --output ../models/

# Train specific model
python train_models.py --dataset ../data/loan_data.csv --model random_forest
```

#### Available Parameters

```bash
--dataset       # Path to training data CSV
--output        # Directory to save trained models
--model         # Specific model to train (optional)
--test-size     # Test split ratio (default: 0.2)
--random-state  # Random seed (default: 42)
```
